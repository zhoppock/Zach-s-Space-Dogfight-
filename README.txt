Welcome to Zachary's Space Dogfight Game!

There are two ships, Yellow and Red.

Yellow moves around with the WASD Keys while Red moves around with the Arrow Keys.

Yellow fires with the LEFT CTRL Key and Red fires with the RIGHT CTRL Key.

First ship to destroy the other wins!
